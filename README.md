# 🪨✂️📄 Rock Paper Scissors (Qt Edition)

Простая и стильная реализация классической игры **"Камень, Ножницы, Бумага"**, написанная на C++ с использованием фреймворка **Qt Framework**. 

## 🎮 Особенности игры
* **Интуитивный интерфейс:** Чистый дизайн, созданный с помощью `QtWidgets`.
* **Рандомный ИИ:** Компьютер делает выбор в реальном времени, используя `QRandomGenerator`.
* **Визуализация:** Отображение выбора игрока и компьютера с помощью иконок.
* **Динамический статус:** Мгновенное определение победителя с цветовой индикацией (Победа/Поражение/Ничья).

---

## 🛠 Технологии
* **Язык:** C++17
* **Фреймворк:** Qt 6.x (поддерживает и 5.x)
* **Сборка:** qmake / CMake
